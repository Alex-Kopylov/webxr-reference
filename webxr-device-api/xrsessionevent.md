# XRSessionEvent

The **`XRSessionEvent`** interface of the of the WebXR API is the event type passed to all of the <a href="xrsession">XRSession<a> events, specifically, `onblur`, `onfocus`, `onresetpose`, and `onend`.

</dl>

## Properties

<dl>

</dl>
